  • 分布式计算系列(I):Yarn基础库初探

    qingyuanxingsi - 六 22 三月 2014 - RPC, YARN, Hadoop, State Machine, Designing Patterns, Reactor Pattern

    上一篇我们比较初步地介绍了一下分布式计算与存储的基础知识，从这一篇开始，我们开始探索分布式计算的相关理论与实践。本系列的若干篇都会从Hadoop 2.0源码出发，拿YARN(Yet Another Resource Negotiator)开刀，以期对分布式计算的基本框架有一个初步的认识。作为分布式计算系列的第一篇，本文会分别介绍Hadoop RPC,事件库以及状态机等基础库,此外,作为一款优秀的开源软件,其设计架构中必然会掺杂很多设计模式，本文会以源码为基础，对其中的部分设计模式展开讨论。
